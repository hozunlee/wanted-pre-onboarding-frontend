# 살아갈 결심 ( To-Do list ) 

우리는 살아가며 수 많은 결심을 합니다. 

살아갈 결심 서비스는 머리 속을 멤도는 수 많은 결심을 정의하고, 결심을 내 손으로 마무리 짓는데 촛점을 맞춥니다.
